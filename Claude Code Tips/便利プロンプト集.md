# 便利プロンプト集

## Codex thin forwarder スキルを作る

任意の作業をCodexに委譲するカスタムスキルを1分で作れるプロンプト。

```
{やりたいこと} をCodexにやらせるSkillを作って。
/codex:rescue の thin forwarder パターンで。
```

**使い方：** `{やりたいこと}` を置き換えて Claude Code に貼るだけ。  
**保存先：** `~/.claude/skills/` に `SKILL.md` が自動生成される。  
**活用例：** 契約書レビュー、要件定義書レビュー、財務計画書レビューなど非エンジニア業務にCodexの論理検証力を転用できる。
