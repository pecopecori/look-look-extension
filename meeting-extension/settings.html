必要な設定が揃うまで録音を開始できません。順番に埋めてください。

  1. 1. ローカル faster-whisper サーバを起動

    ターミナルで cd /Users/demo/開発/meeting-extension/local-server && ./install.sh を1回実行。Macログイン時に自動起動するようになります。
    動作確認: curl http://127.0.0.1:9000/health

  2. 2. Gemini API キーを取得（無料）

    Google AI Studio で「Create API key」→ コピー。下のフォームに貼り付けてください。無料枠は 1分15req / 1日1500req まで。

  3. 3. Google Cloud で OAuth Client ID

    Docs / Drive / Sheets / Calendar API を有効化したプロジェクトで「Chrome 拡張機能」種別の OAuth Client ID を発行し、manifest.json の oauth2.client_id に貼り付け→拡張機能を再読み込み。下の「認証して Drive / Sheets を準備」を押すとフォルダと台帳が自動で作られます。
